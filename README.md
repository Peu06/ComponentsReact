# ComponentsReact

Projeto de exemplo em React com componentes reutilizáveis.

## Pré-requisitos

Antes de começar, verifique se você tem instalado:

- [Node.js](https://nodejs.org/) (versão recomendada >= 16)
- npm (geralmente instalado junto com o Node.js)

# Instalando e executando o projeto

## Clone o repositório:

```bash
git clone https://github.com/SEU-USERNAME/ComponentsReact.git
cd ComponentsReact
```

## Instale as dependências:
```bash
npm install
```

Isso vai instalar todas as dependências listadas no package.json, incluindo react-scripts.

## Executando o Projeto
```bash
npm start
```
O projeto será aberto automaticamente no navegador em http://localhost:3000
Qualquer alteração no código será refletida imediatamente no navegador.
